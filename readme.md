# Rogue Fable IV Localization

로그 페이블 iv 게임의 현지화 패치 툴킷입니다.

로그 페이블 IV는 현재 영문만 지원합니다.

이 리포지토리는 다음을 수행합니다

- i18n 지원
- 커스텀 폰트 및 출력 (rich text 호환.)
- (한국어 유저에게만 해당) 한국어 기계 번역 지원
- 다른 언어를 쉽게 추가할 수 있습니다.
